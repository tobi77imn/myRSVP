# myRSVP

Previously created this RSVP for my wedding, to track the attendance of guests. This helps  to firsthand coordinate with floor manager to ensure the food and crowd were sufficient and no hiccups during the event.

The link for the RSVP > https://tinyurl.com/RSVP-ImanandJannah
